# Modern C++ (C++20+)

Examples demonstrating modern C++ features and idioms: Concepts, Ranges, Coroutines, `std::format` / `<=>` and atomics.

Build using the included `CMakeLists.txt` inside this folder (requires CMake >= 3.20 and a compiler supporting C++20/23).

Key files: `Concepts_Example.cpp`, `Ranges_Example.cpp`, `Coroutines_Simple.cpp`, `FormatAndSpaceship.cpp`, `AtomicsMemoryModel.cpp`.
